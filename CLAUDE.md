# 注意事項
- isabelle は /isbman という isabelle プロセスマネージャがあるのでそれを使って操作して下さい。
- 現在同じマシン上で２つの証明エージェントが別の証明をしていますので、kill等は isbmanを通して行ったり、気をつけてください。
- prss-proof/, pss-proof/ 以下は触らないでください。（他のエージェントが作業中)
- .git は git/.git にあります。
- git worktree は <worktree-name>/ で作成して下さい.
- ユーザーのターンのときは文章の最後に👤を、ツール待ちorエージェント待ちの時には文章の最後に🤖をつけてください。
- AskUserQuestion ツールは使わず、直接チャットで質問して下さい。（AskUserQuestion ツールを使うと捨てた選択肢が消えて見えなくなる）

# 自律性
- できるだけ自律的に方針を選択し、ユーザーの判断を仰ぐことなく自律的に証明完成まで進んでください。
- ユーザーにターンを回してよいのは、どうしても方針の相談をしたいときに限ります。
  - 「進めてよいですか？」というだけの質問は禁止です。そのような質問をする際には質問をせずに進めてください。
  - ユーザーのターンを作る目的で無理やり方針の相談を作らないでください。
  - compact は 100% になると自動的に行われるのでその方法で行ってください。ユーザーに compact を要求しないでください。
- セッションの長さやトークンの長さは気にしないでください。どれだけ長くなってもどれだけトークンを使っても問題ありません。それを理由に作業を止めないでください。
- 本リポジトリへのgit commit と git push は自由です。許可なく行ってください。
- 選択肢が１つしかない場合は、ユーザーの指示を待たずに自動的に選択してください。
- できるだけ自走して下さい。ユーザーの指示を待たない。
